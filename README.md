## ğŸ§  How the AI Agent Works

### Architecture Flow

```
CORRECT FLOW (makes sense!):

1. BEFORE GAME: Agent analyzes past performance
   - Identifies weak topics (<60% accuracy)
   - Identifies strong topics (>80% accuracy)
   - Calculates recommended difficulty

2. BEFORE GAME: Agent generates 50 personalized questions
   - 60% focus on weak topics
   - 30% mixed/exploratory topics
   - 10% hard questions on strong topics

3. DURING GAME: User plays with those personalized questions
   - Frontend tracks each attempt
   - Records correct/wrong answers
   - Measures time spent

4. AFTER GAME: Submit results to backend
   - Save game session stats
   - Update topic performance
   - Agent learns from this session

5. NEXT GAME: Agent generates EVEN BETTER questions
   - Based on what you just played
   - Adapts to your new performance
   - Cycle repeats - agent keeps learning!
```

### Memory & Context

The agent maintains:
- **Historical Performance**: All question attempts stored
- **Topic Proficiency**: Accuracy per SAT topic
- **Learning Patterns**: Time spent, difficulty progression
- **Context Memory**: Understands learning journey over time

### Step 2: Update Frontend to Use API

Create `frontend/lib/api.ts`:

```typescript
const API_URL = 'http://localhost:8000'

export async function getPersonalizedQuestions(userId: number, numQuestions: number) {
  const response = await fetch(`${API_URL}/questions`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ user_id: userId, num_questions: numQuestions })
  })
  return response.json()
}

export async function submitGameResults(results: any) {
  const response = await fetch(`${API_URL}/game-results`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(results)
  })
  return response.json()
}

export async function getInsights(userId: number) {
  const response = await fetch(`${API_URL}/insights/${userId}`)
  return response.json()
}
```

### Step 3: Use in Game Components

```typescript
// In ZombieGameContainer.tsx
import { getPersonalizedQuestions, submitGameResults } from '@/lib/api'

// On game start

  user_id: userId,
  game_type: 'zombie',
  score: gameState.score,
  // ... other stats
})
```

### Adding a New Game

1. Create game folder: `games/[game-name]/`
2. Add types: `types.ts`, questions: `questions.ts`, game: `[GameName]Game.ts`
3. Create container: `components/[GameName]GameContainer.tsx`

- Agent works best with 10+ question attempts
